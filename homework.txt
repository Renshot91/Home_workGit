Задача 1.Как скопировать на свой компьютер репозиторий из github.com, что 
должно быть сделано предварительно?

2.Скопировать URL репозитория
3.Открыть терминал 
4.Через команду cd  перейти в каталог где нужно сохранить репозиторий
5.Клонировать с помощью команды git clone 
Пример
Git clone git@github.com:Renshot91/Home_workGit.git

Задача 4.Какую информацию нам дают команда :
git status - показывает состояния файлов в рабочем каталоге и индексе
git log - используется для просмотра истории коммитов в репозитории Git.
git remote -v - используется для просмотра информации о удаленных репозиториях, связанных с вашим локальным Git-репозиторием
