Задание 1.Как скопировать на свой компьютер репозиторий из github.com, что 
должно быть сделано предварительно?
1.Создать репозиторий на GitHub(Home_workGit)
2.Скопировать URL репозитория
3.Открыть терминал 
4.Через команду cd  перейти в каталог где нужно сохранить репозиторий
5.Клонировать с помощью команды git clone 
Пример
Git clone git@github.com:Renshot91/Home_workGit.git
